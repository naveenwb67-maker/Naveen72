# My UID Tool (Vercel-ready)

This is a small React + Vite + Tailwind project scaffold that contains the UID Editor UI.

## How to use

1. Install dependencies:
   ```
   npm install
   ```

2. Run locally:
   ```
   npm run dev
   ```

3. Build:
   ```
   npm run build
   ```

## Deploy to Vercel

- Push this project to GitHub.
- Import the repo in Vercel (https://vercel.com) and deploy. Vercel detects Vite and runs `npm run build` by default.
